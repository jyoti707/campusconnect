# CampusConnect

A MERN Stack Student Activity Management Portal.

## Features

- Student Registration
- Login Authentication
- JWT Authorization
- Dashboard Access
- Task Management
- Search Students
- Filter Tasks
- MongoDB Integration

## Technologies

- React
- Node.js
- Express.js
- MongoDB
- JWT
- Docker

## Run Project

Backend:

npm install
npm run dev

Frontend:

npm install
npm run dev
